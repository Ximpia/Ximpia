Get The Code : Ximpia Software Repositories
Our software is open source with Mozilla public license. You can get code at GitHub at the following repositories:

Ximpia/core-django
Heart of framework with common python code like workflow, register of views and actions, core model, etc... for django framework

Ximpia/django
Main project with django code and site applications.

Ximpia/front
Front software, like css, images, media, javascript code and jQuery ximpia plugins.
